EC2 Instance Challenge Lab

Project Overview

This project involves setting up an EC2 instance, configuring a web server, and deploying

Steps to Complete

  • Launch an EC2 instance on AWS
  • SSH into the instance and install the necessary software
  • Configure the web server and deploy the web page.
  • Verify the web page is accessible over the internet.

Conclusion

This project is a great introduction to using AWS EC2 and managing a Linux-based web server. It provides a practical understanding of cloud computing and the foundational skills needed to manage cloud infrastructure.
